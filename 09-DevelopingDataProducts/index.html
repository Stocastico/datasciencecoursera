Classify Iris flowers using an SVM model

The idea of the application is to experiment with two SVM hyperparameters, the cost and the kernel, and see how they affect the accuracy of the classification.

The dataset used for the project is the classic iris dataset available in R.

Once the user has chosen the parameters, he can then click the button and see how accurate the classification was, as well as see which elements in the dataset were classified properly and which were not.

IRIS dataset

The dataset used contains information of 150 elements. For each element, information about the Sepal width and length and about the Petal width and length is provided. For each flower, the columns Species gives the info about the correct species of the flower.
